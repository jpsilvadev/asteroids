# asteroids

A clone of the arcade Asteroids game built with Pygame

## Prerequisites
- **Ensure you have Python installed**
```bash
$ python3 --version
```
- Create a virtual environment
```bash
$ python3 -m venv venv
```
- Activate the venv:
```bash
$ source venv/bin/activate
```
- Install the dependencies from `requirements.txt`:
```bash
$ pip install -r requirements.txt
```

## Getting Started
1. Clone the repo:
```bash
$ git clone https://github.com/jpsilvadev/asteroids
$ cd asteroids
```
2. Run the game:
```bash
$  python3 main.py
```

- Use `WASD` for movement and `space` to shoot
